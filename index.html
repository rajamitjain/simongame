This is a Memory Game


The device creates a series of tones and lights and you should be able to repeat the sequence. The device has four colored buttons, each producing a particular tone when it is pressed or activated by the device.
A round in the game consists of the device lighting up one button in a random order,
after which the player must reproduce that order by pressing all the buttons from the beginning.
As the game progresses, the number of buttons to be pressed increases.


The Level Indicates The Number of colors or tiles In the sequence. Remember the order
